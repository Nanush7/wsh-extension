Thank you for using the WCA Staff Helper extension. Here you have some instructions to use it:

Shortcut Action
Ctrl+Shift+Space Link the selection with the long version of the item (i.e. WSAP => WCA Scramble Accountability Policy).
Ctrl+Shift+Comma Link the selection with the short version of the item (i.e. wsap => WSAP).
Ctrl+Shift+Period If you have selected a regulation or a guideline, it will be displayed in a popup.

You can change the key combinations by navigating to chrome://extensions/shortcuts (be careful to avoid conflicts with other extensions).
